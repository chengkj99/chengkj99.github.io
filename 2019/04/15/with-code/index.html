关于代码的那点思考

2019-04-15
思考

有一段时间认为，写代码是和机器打交道，写久了会忘记如何与人打交道。现在的想法有点改变了：你必须对你的代码实在，用心，这样代码才能够很好的运行，你也不用为改 bug 而烦恼，就像是对身边的人一样如此。
