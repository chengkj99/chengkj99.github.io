技术

什么是 babel

babel 是 javascript 编译器，它可以将 ECMAScript 2015+ 代码转换成浏览器可解析的兼容性代码。

babel 是基于 plugin system 进行代码转换的，通过 babel 每个人都可以写自己的 babel plugin。

什么是抽象语法树 Abstract Syntax Tree (AST)


AST (Abstract syntax tree, 抽象语法树) 的学习资料：

  • Leveling Up One’s Parsing Game With ASTs by Vaidehi Joshi * (Highly recommend this one! 👍)
  • Wikipedia’s Abstract syntax tree
  • What is an Abstract Syntax Tree by Chidume Nnamdi


简单说来：AST 是代码的树形表示，该树形表示遵循 estree specification 规范。


AST 能表示代码的结构和含义，所以他允许像 babel 这样的编译器去理解和转换它。

在继续之前，让我们看一下 Identifer（path）{} 的参数。它的参数是路径而不是节点，路径和节点之间有什么区别？
在 babel 中，path 是 node 上的抽象，它能表示节点之间的链接，比如：通过父节点可以找到子节点等等，它还能代表 scope 域、context 上下文等信息，此外 path 提供了 replaceWith, insertBefore, remove 等方法，这些方法的操作将更新 AST 中对应的节点。
